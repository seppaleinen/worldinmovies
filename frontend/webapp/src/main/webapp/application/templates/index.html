Welcome to the world in movies

This website started as I was curious about what parts of the world I'd seen movies from and how many countries that I hadn't seen any movies from
So I started on making this, which should be able to help in visualizing the answer to that question
And hopefully it'll be interesting seeing the results


For now there's only two pages working
  • Map - Shows what countries you've seen movies from, and lists top rated movies from every country
  • Chart - Shows how many movies you've seen from each country, in a pie-chart
